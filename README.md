# Ansible Role: marvel-nccr.raspa

An ansible role that installs [RASPA2](https://github.com/numat/RASPA2) on Ubuntu.

## Installation

`ansible-galaxy install marvel-nccr.raspa`

## Role Variables

See `defaults/main.yml`

## Example Playbook

  - hosts: servers
    roles:
    - role: marvel-nccr.raspa
